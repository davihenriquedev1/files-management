Galeria de Fotos:

Página Carrega -> Lista os álbuns de foto -> 
Cada álbum é um link dinamico (album/1...)
Ao clicar no album ele vai pra rota do álbum com o título do álbum e respectivas fotos
Ao clicar na foto ele abre a foto, a qual é uma rota (photo/23)

Exigências:

1. Usar Rotas com react router
2. Usar Rotas em Objeto

API exemplo: 
url/albums
url/albums/1
url/albums/1/photos
url/photos/1
